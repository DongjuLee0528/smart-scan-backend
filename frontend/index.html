자동 스캔

현관 통과만으로 소지품 자동 체크. 주머니나 가방 안에 있는 태그도 완벽하게 인식하여 따로 꺼낼 필요가 없습니다.

실시간 알림

누락 물품 즉시 이메일 알림. 필수 소지품을 챙기지 않고 문 밖을 나서는 순간, 설정된 이메일로 경고 알림이 발송됩니다.

가족 관리

가족 구성원별 개인 소지품 관리. 대시보드에서 각자의 물품을 등록하고 누구의 물건이 누락되었는지 한눈에 파악하세요.

작동 방식

단 4단계로 완벽한 소지품 체크

1. RFID 태그 부착

관리할 소지품에 소형 태그를
부착합니다.

2. 현관 통과

외출 시 평소처럼
현관을 나섭니다.

3. 자동 스캔

리더기가 부착된 모든
태그를 즉시 스캔합니다.

4. 누락 알림

필수 소지품이 없을 시
이메일로 즉시 알려줍니다.

지금 바로 소지품 걱정 없는 일상을 시작하세요

단 1분의 회원가입으로 가족 모두의 평화를 지킬 수 있습니다.
